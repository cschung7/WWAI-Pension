# 한국 IRP/DC 퇴직연금 규제 프레임워크
**Updated**: 2026-03-30 | Source: 근로자퇴직급여보장법, 퇴직연금감독규정, FSC 혁신금융서비스 지정

---

## 1. 계좌 유형

| 유형 | 정의 | 특징 |
|------|------|------|
| **DB형** (확정급여형) | 기업이 운용, 퇴직 시 사전 확정 급여 | 개인 운용 불가 |
| **DC형** (확정기여형) | 기업이 납입, 개인이 직접 운용 | 개인 ETF 선택 가능 |
| **IRP** (개인형 퇴직연금) | 누구나 가입 가능, 세액공제 핵심 수단 | 직장인+자영업자+공무원 |

**WWAI-Pension 타깃**: DC형 + IRP (개인이 운용 지시하는 계좌)

## 2. 납입 한도 및 세액공제

| 항목 | 내용 |
|------|------|
| **연간 납입 한도** | IRP + DC 합산 1,800만원 |

| **세액공제율 (중저소득)** | 총급여 5,500만원 이하 → 16.5% → 최대 **148.5만원** 환급 |
| **적립금 이월** | 미사용 로보어드바이저 일임 한도는 다음 해로 이월 가능 |

## 3. 위험자산 한도 (핵심 규제)

**근로자퇴직급여보장법 시행령 제25조, 퇴직연금감독규정 제18조**

```
위험자산(주식형) ≤ 전체 적립금의 70%
안전자산(채권·MMF) ≥ 전체 적립금의 30%
```

### 위험자산 (IRP 내 최대 70%)
- 국내 주식형 ETF (KOSPI200, 코스닥150 등)
- 해외 주식형 ETF (S&P500, 나스닥100, 일본/인도/중국 등)
- 국내 테마형 ETF (반도체, 이차전지, 방산 등)
- 원자재/금 ETF (금현물, 은선물 등)

### 안전자산 (최소 30% 필수)
- 채권형 ETF (국고채, 회사채 AA- 이상)
- 머니마켓 ETF (CD금리, KOFR, MMF)
- 혼합채권형 ETF (주식 혼합 비율 40% 이하)

### 완전 금지 상품
| 금지 유형 | 예시 |
|-----------|------|
| 레버리지 ETF (2x, 3x) | KODEX 레버리지, TIGER 200선물레버리지 |
| 인버스 ETF (-1x, -2x) | KODEX 인버스, KODEX 200선물인버스2X |
| ELS/DLS | 원금비보장 구조화 상품 |
| 파생결합증권 | 일부 합성 ETF (단, 금리형 합성 ETF는 허용) |

> ⚠️ 커버드콜 ETF: 현재 IRP 허용 여부 회색지대 — 보수적으로 제외

## 4. 로보어드바이저 일임 서비스 (2025-03-27 시행)

**FSC 혁신금융서비스 지정** (금융위원회 2024-12 지정)

| 항목 | 내용 |
|------|------|
| **최초 출시** | 파운트투자자문 + 하나은행 (2025-03-27) |
| **4월 출시 예정** | 미래에셋자산운용, 한투운용, 쿼터백, 디셈버앤컴퍼니, 미래에셋증권, NH투자증권 |
| **연간 일임 한도** | IRP 계좌당 900만원 |
| **한도 이월** | 미사용분 다음 해 이월 가능 (예: 1년차 500만원 사용 → 2년차 한도 1,300만원) |
| **복수 이용** | 여러 로보어드바이저 업체 합산 한도 900만원 이내 |
| **직접운용 병행** | 한 IRP 계좌에서 직접운용 + 로보어드바이저 혼합 가능 |
| **알고리즘 검증** | 코스콤 테스트베드 심사 필수 |

| 위험등급 | 허용 상품 유형 | WWAI 매핑 |

| 중위험 | TDF, 밸런스드펀드 | 안전자산 50% + 위험자산 50% |
